# Topic
Building AI-powered workflows with Claude CoWork for executives and knowledge workers.

# Speaker
Alex Lieberman & JJ Englert

# Source
https://www.youtube.com/watch?v=Qteg_2HFvZ0

# Summary

The discussion explores how Claude CoWork can function as an AI chief of staff by combining workspace instructions, connected applications, and reusable skills.

JJ Englert demonstrates how users can build structured AI systems using folder hierarchies, markdown skills, global instructions, and workspace maps. These systems allow AI to act as a CEO coach, advisory board, content assistant, and productivity partner.

The episode emphasizes that effective AI usage is no longer about isolated prompts but about building repeatable workflows, persistent context, and reusable systems.

# Key Ideas

## 1. AI works better with persistent context

Chat sessions lose context, while CoWork uses:
- Global instructions
- Workspace maps
- Folder structures
- Skills
- Documentation

This allows AI to remember preferences, goals, workflows, and writing styles.

## 2. Skills are reusable prompts

A skill is essentially a markdown document that defines:
- Instructions
- Processes
- Examples
- Desired outputs

Skills create consistent results across tasks.

## 3. Folder organization improves AI performance

Projects should contain:
- Instructions
- Brand voice
- Analytics
- Examples
- Successful outputs

This gives AI access to the information required to perform specialized tasks.

## 4. AI can function as an executive coach

Custom skills can help:
- Evaluate decisions
- Analyze tradeoffs
- Identify risks
- Provide strategic guidance

## 5. Advisory boards can be simulated with AI agents

Multiple AI personas can evaluate:
- Customer perspective
- Operations
- Finance
- Risk
- Leadership

This creates a structured decision-making process.

## 6. Marketing requires systems thinking

The episode argues that modern professionals must combine:
- Domain expertise
- AI capabilities
- Workflow design
- Feedback loops

This emerging role resembles the "Marketing Engineer."
